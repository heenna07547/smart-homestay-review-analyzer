# Smart Homestay Review Analyzer

## Overview

Smart Homestay Review Analyzer is a full-stack web application designed to help homestay owners understand and manage guest feedback more effectively. Homestays often receive reviews from different booking platforms and social media channels, making it difficult to manually analyze customer opinions and identify recurring issues.

This project uses Artificial Intelligence to automatically analyze guest reviews, determine sentiment, detect common complaints, and generate useful insights. The goal is to help property owners improve customer satisfaction and make data-driven decisions based on guest experiences.

---

## Problem Statement

Homestay owners spend a significant amount of time reading guest reviews to understand customer satisfaction. Important feedback may be missed when dealing with a large number of reviews.

This application aims to simplify the review analysis process by providing automated sentiment analysis, complaint detection, and visual reporting tools.

---

## Features

### Review Collection

Store and manage guest reviews from multiple sources in a centralized platform.

### Sentiment Analysis

Analyze reviews and classify them as Positive, Negative, or Neutral using AI-powered text processing.

### Rating Dashboard

Display review statistics, sentiment distribution, and overall customer satisfaction through interactive charts and graphs.

### Complaint Detection

Identify common issues related to cleanliness, food quality, service, location, facilities, and other important factors.

### Report Generation

Generate summarized reports that help owners quickly understand customer feedback and take corrective actions.

---

## AI Integration

This project uses the OpenAI API for Natural Language Processing (NLP) tasks.

The AI module will:

* Analyze customer sentiments
* Detect common topics and complaints
* Summarize large collections of reviews
* Generate useful insights from guest feedback

Using OpenAI enables accurate language understanding and improves the quality of review analysis.
